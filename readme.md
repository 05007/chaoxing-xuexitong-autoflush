用法:

---

① 
```
git clone https://github.com/ZhyMc/chaoxing-xuexitong-autoflush
```
② 
```
cd chaoxing-xuexitong-autoflush   (到程序目录底下)
```
③
```
npm install
```

之后你就可以通过执行 

```
npm start 
```

启动本程序

---

说明：
```
	本程序会自动遍历你的所有课程，课程的所有章节，甚至包括上锁的章节内的视频都会被自动刷完。
所以你可以挂机本程序至后台。因为由 Node.js 平台编写，你可以在Linux下运行本程序.
```
